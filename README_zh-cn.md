## 背景

vue-ele 是一个仿造饿了么外卖的源码。它基于 Vue.js 3 和 TypeScript 实现，整体 UI 模仿饿了么外卖。它使用了最新的前端技术栈，包括 Vite、Vue Router、Pinia 等。

## 安装

```shell
# 克隆项目
git clone git@github.com:GeniusCorn/vue-ele.git

# 进入项目目录
cd vue-ele

# 安装依赖
pnpm

# 开发
pnpm dev

# 构建
vite build
```
